Totally Awesome CSS Library

This Library was created as a school project.

These elements were modified:

  • All <h> elements
  • <p> element
  • <ul>, <ol> and <li> elements
  • <b> and <strong> elements
  • Remade <code> element
  • Remade <a> element

To acces this part of Library use this file.

Components created in this Library:

  • .codeblock component to show full sections of code
  • .dbutton component to indicate download button
  • modular .card component
  • responsive .image component
  • custom .navbar component

To acces any of these components use this file.

Code definition:

<code> element can be used as part of a text:

Lorem ipsum dolor sit amet consectetur, printf("Hello, world!"); adipisicing elit. Beatae soluta, modi totam saepe ratione dignissimos eligendi quaerat fugit accusamus a id ex quisquam nobis perspiciatis minus repellat praesentium porro impedit!

or as codeblock using this structure:

<div class="codeblock"> <code class="codeblock__content"> #include <stdio.h> int main(void) { printf("Hello, world!") } <code> <div>

You have to put all your code at the begining of the line to avoid createing whitespace
